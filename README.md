# Jekyll Img Prefix Filter

## Installation

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "jekyll-img-prefix"
```

Add `jekyll-img-prefix` to your Jekyll site's `_config.yml`:

```yaml
plugins:
  - jekyll-img-prefix
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll-img-prefix
